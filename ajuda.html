Como jogar
O objetivo do bingo é preencher uma cartela. Os números devem ser preenchidos conforme são sorteados. Quando a cartela for preenchida, o jogador deve dizer "bingo" e o sorteio deve ser interrompido para verificar se todos os números da tabela foram sorteados, de forma a confirmar a vitória.
Sortear manualmente
Clique no botão com a seta para a direita para sortear um número, ou use o botão direcional direito do teclado. Para começar um novo jogo, clique no botão com a seta circular.
Sortear automaticamente
Clique no botão "play" para sortear números automaticamente, com o intervalo de tempo determinado pelo valor selecionado na lista. Quando um jogador estiver por apenas um número, o modo automático deve ser substituido pelo modo manual, para não interferir na validação de vitória. Clique no botão "pausa" para interromper o sorteio automático. Para começar um novo jogo, clique no botão com a seta circular.
Campo de informações
Apresenta a quantidades de números restantes e quantas partidas foram jogadas.
Validação de vitória
Quando um jogador fizer bingo. clique no botão "Listar Números" e verifique na lista ao lado se algum número ali presente está na cartela do jogador. Se não estiver, ele de fato venceu. Caso contrário, ele não venceu, e o jogo pode proceguir.
